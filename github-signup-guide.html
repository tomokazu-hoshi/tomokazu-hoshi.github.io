GitHubアカウント作成 完全ガイド

GitHubは世界中のクリエイターが利用するプラットフォームです。サイト全体は英語ですが、手順は非常にシンプルです。ここでは、PCとスマホ（iPhone/Android）それぞれの登録手順を、迷わないように解説します。

📝 登録前に用意するもの

  • メールアドレス： 認証コードを受け取れるもの（Gmail推奨）
  • 希望のユーザー名： あなたのサイトのURL（ユーザー名.github.io）の一部になります
  • パスワード： 8文字以上、英数字を含むもの

1. PCでの登録手順

  1. GitHub公式サイトのサインアップページを開きます。
  2. Email address: アドレスを入力し「Continue」を押します。
  3. Password: パスワードを決めて「Continue」を押します。
  4. Username: ユーザー名を入力します。
    • ※URLの一部になるため、シンプルで分かりやすい名前がおすすめです。
  5. Email preferences: お知らせが不要なら「n」と入力して「Continue」。
  6. Verify your account: パズルを解いて人間であることを証明します。
  7. メールに届いた **8桁の認証コード** を入力すれば完了です。

2. スマホ（iPhone/Android）での登録手順

※専用アプリを入れなくても、ブラウザ（SafariやChrome）から登録可能です。

  1. スマホのブラウザで GitHubサインアップ画面 にアクセスします。
  2. 画面の指示に従い、メールアドレス ➔ パスワード ➔ ユーザー名を順に入力します。
  3. 「Create account」をタップし、パズル認証を完了させます。
  4. 届いたメールのコードを入力すれば設定完了です。
⚠️ 重要なアドバイス

GitHub Pagesを利用する場合、独自ドメインを設定するまでは 「https://ユーザー名.github.io/」 というアドレスがあなたのサイトの顔になります。 覚えやすく、サイトの内容に関連したユーザー名を設定しましょう。
